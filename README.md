## Rediminds Chatbot application for IOS and Android

### Getting started

- Clone repo
- cd into chatbot-mobile-app/
- npm install
- cd into ios/
- pod install
- cd ..
- npm run ios
- npm run android

### Prerequisities

- XCode
- Android Studios
